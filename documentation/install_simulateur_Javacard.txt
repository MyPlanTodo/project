Environnement :
Une façon relativement simple de développer des applets pour des cartes
Javacard est d'utiliser Eclipse, disponible dans les dépôts. 

Il est nécessaire de créer ou compléter quelques variables d'environnement :
JAVA_HOME=<chemin vers le jdk 1.5>
JC_HOME=<chemin vers le javacard_dk 2.2.2>
PATH=$PATH:$JAVA_HOME/bin:$JC_HOME/bin

La dernière version du SDK Javacard pour GNU/Linux est la 2.2.2. Cependant,
cette version utilise le JDK 1.5, d'où la version indiquée dans $JAVA_HOME.
Les binaires du "$JC_HOME/bin" permettent, entre autres, de tester une
applet via un simulateur. La méthode ou l'API utilisée ne sera cependant pas 
décrite ici. Un tutoriel est disponible à l'adresse suivante :
http://julienb.developpez.com/tutoriels/java/introjavacard/

-téléchargement du Javacard Kit 2.2.2
-décompression dans un dossier

-téléchargement du plugin Eclipse JCDE
