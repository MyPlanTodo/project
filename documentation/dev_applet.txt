Pré-requis :
L'environnement, le JDK et la configuration de l'applet sont les mêmes que pour
le développement d'une applet avec simulateur.

Concernant l'application cliente, un exemple est disponible :
samples/client_gen_random.java. Notons que contrairement à l'applet, il est
possible d'utiliser une version de Java en version supérieure à 1.5.

L'installation de cette applet se fera en suivant les instructions du fichier
./installation_applet.txt
